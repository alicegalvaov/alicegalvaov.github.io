Selected Projects

01 / Featured Project
What Skills Does Your Region Actually Want? A Big Data Analysis of the U.S. Data Job Market

Which data skills are most in demand, and does the answer change depending on where you live? I collected 3,900 real job postings across 335 U.S. cities using the JSearch API, built a full cloud pipeline with PySpark and Snowflake using a Bronze, Silver, Gold architecture, and applied NLP to extract technical skills from job descriptions. The result is a regional skills map showing that Tableau dominates the West, Power BI leads the South and Midwest, and California, Texas, and New York each have a different most-wanted data role.

✦ 3,900 job postings · 335 cities · 23 states · Full ETL pipeline from API to Tableau dashboards
Python PySpark Snowflake Tableau NLP ETL Pipeline JSearch API Big Data

02
Airbnb Profit Maximization Using an Occupancy Rate Regressor

Can machine learning tell an Airbnb host what price maximizes their yearly profit? We built a deep neural network to predict occupancy rate from 98,000+ listings across Los Angeles, Austin, and New York City, then used that model to sweep a price range and find the profit-maximizing price for any given listing.

✦ Neural network R² of 0.46 · Optimization curve identifying price that projected $77,992 max yearly profit
